- Тип тюнера: DSB
- Диапазоны: УКВ/ДВ/КВ1-6
- Частоты для каждого диапазона: УКВ (64-108МГц) / ДВ (150-280КГц) / КВ1-6 (6,0-18МГц)
- Мощность динамиков: Φ57mm/8Ω/1W
- Питание: АА*4шт 6V
- Размер корпуса: 150х42х92мм
